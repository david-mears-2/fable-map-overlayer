# London Housing Heat Map

Interactive heat map of Greater London scored on housing desiderata. Full design in
[london-heatmap-spec.md](london-heatmap-spec.md).

**Status: vertical slice** — map + heat layer over dummy data, with per-layer weight
sliders, solo toggles, and a global heat-opacity control. The data artefact is
generated by a stand-in script that follows the real schema (spec §4); the Python
pipeline (`pipeline/`) replaces it later without frontend changes.

## Run

```sh
cd web
npm install
npm run generate   # writes public/grid.json (dummy, deterministic)
npm run dev        # http://localhost:5173
```

`npm run build` type-checks and produces `web/dist/` for GitHub Pages.

## Layout

- `web/` — Vite + TypeScript + MapLibre GL frontend (no framework, spec §8)
- `web/scripts/generate-dummy-grid.mjs` — dummy artefact generator (250 m BNG grid,
  reprojected to WGS84; winsorized-linear scoring per spec §7)
- `pipeline/` — (not yet built) offline data pipeline, spec §9
